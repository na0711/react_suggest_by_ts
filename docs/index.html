React サジェスト（use Typescript）

概要

  • 都道府県名をひらがな、ローマ字表記からサジェストします。
  • サジェストリストは、マウススクロール、クリックで操作可能です。
  • さらに、上下キー、エンターキーで操作可能です。

目的

下記2点を学べる課題として、サジェストを作成。

  • reactの基礎を学ぶために、親の入力内容に応じて子コンポーネントが動的にかわるコンポーネント。
  • 親コンポーネントだけでなく、子コンポーネントもユーザーの操作対象になるコンポーネント。

はまったポイント

  • 上下キーでサジェストリストのフォーカスを操作する際、this.state.focusの更新直後のスクロール量算出がうまくいかず試行錯誤。componentDidUpdateで解決。
  • 途中まで、ReactDOM.findDOMNode()で実装してしまう。refに移行しようとしたものの、うまく機能せず試行錯誤。公式ドキュメントの「Refs and the DOM」を読んで解決。
  • typescriptでclickのeventの型がわからないまま
